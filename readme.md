## ✨ Overview

GPTPrint Pro is a free Chrome extension that allows users to:

- 🖨️ Print ChatGPT conversations  
- 📄 Export ChatGPT chats to PDF  
- 🎨 Fully customize print layout  
- 💧 Add watermark & branding  
- 📐 Control margins, fonts & alignment  

Designed for students, developers, teachers, researchers, and professionals.

---

## 🎨 Features

### 🖨️ Clean PDF Export
- A4 optimized layout  
- Portrait & Landscape modes  
- Remove unnecessary UI clutter  

### 🎨 Theme Control
- Light mode  
- Dark mode  
- Professional formatting  

### 🧑‍💻 Message Customization
- Customize user & assistant message styles  
- Control alignment  
- Hide selected messages  
- Rounded corner styling  

### 📄 Header & Branding
- Custom title  
- Subtitle support  
- Logo support  
- Adjustable font size  

### 💧 Watermark Support
- Custom text watermark  
- Adjustable opacity  
- Position & rotation control

## 🛠️ Installation (Development)

1. Clone this repository
2. Open Chrome → Extensions
3. Enable Developer Mode
4. Click “Load Unpacked”
5. Select project folder
